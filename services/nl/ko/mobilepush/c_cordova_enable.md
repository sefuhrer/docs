# 푸시 알림을 수신하도록 Cordova 애플리케이션 설정
{: #cordova_enable}

Cordova는 JavaScript, CSS 및 HTML을 사용하여 하이브리드 애플리케이션을 빌드하는 플랫폼입니다.
{{site.data.keyword.mobilepushshort}}는 Cordova 기반 iOS 및 Android 애플리케이션의 개발을 지원합니다. 

Cordova 애플리케이션이 푸시 알림을 수신하고 사용자 디바이스에 푸시 알림을 전송하도록 설정합니다.
